# 🛵 Tracker Delivery

Uma aplicação web desenvolvida para auxiliar entregadores a registrar e calcular o faturamento financeiro diário de forma rápida e eficiente.

## 🚀 Funcionalidades
- Registro individual do valor de cada entrega em tempo real.
- Armazenamento do histórico de corridas do turno (na memória).
- Cálculo automático do faturamento total da noite.
- Interface limpa em "Dark Mode" para não cansar a visão durante o trabalho.

## 🛠️ Tecnologias e Conceitos Utilizados
- **HTML5 e CSS3:** Estruturação e estilização da interface.
- **JavaScript (Vanilla):** - Manipulação do DOM (Eventos de clique).
  - Estruturas de dados (Arrays).
  - Laços de repetição (`for`).
  - Lógica de validação (`if/else` e `return`).
